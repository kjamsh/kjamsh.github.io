I'm a software developer passionate about the interactions between systems and algorithms for big data processing, especially on graphs. You can find my talks on YouTube.

I built Peregrine, a fast and programmable graph mining system which analyzes user queries to aggressively optimize execution.

I'm currently working on hard problems in fault tolerance and distributed stream processing.
